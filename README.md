# 🔒 ScreenLock

A modern Android Screen Lock application built with Kotlin, providing multiple lock styles and customization options for enhanced security and personalization.

## ✨ Features

### 🔐 Security Options

* PIN Lock
* Pattern Lock
* Zipper Lock
* Zipper + PIN Lock
* Zipper + Pattern Lock

### 🎨 Customization

* Multiple zipper styles
* Custom zipper handles
* Custom lock screen wallpapers
* Personalized lock screen experience

### 🖼️ Wallpapers

* Browse and apply beautiful wallpapers
* Powered by the Pexels API
* Download and set wallpapers directly from the app

### 📱 User Experience

* Modern Material Design UI
* Smooth animations
* Fast and lightweight
* Easy setup and configuration

## 🛠️ Built With

* Kotlin
* Android SDK
* MVVM Architecture
* Jetpack Components
* Coroutines
* Retrofit
* Pexels API


## 🚀 Getting Started

### Prerequisites

* Android Studio Hedgehog or newer
* Minimum SDK: 24+
* Kotlin

### Installation

```bash
git clone https://github.com/jehanzebali44/ScreenLocker.git
```

Open the project in Android Studio and run it on your device or emulator.
